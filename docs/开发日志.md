## v83 (2026-07-04)

### Bug修复
1. **Lottie CDN被浏览器拦截**：Edge浏览器的Tracking Prevention会阻止cdnjs.cloudflare.com的Lottie库。改为本地化引用（assets/lottie.min.js），彻底解决CDN依赖问题。
2. **stone-lion.jpg HTTP2协议错误**：GitHub Pages的HTTP/2传输该文件时报错。重新压缩JPEG（268KB→229KB），移除可能导致问题的元数据。
3. **favicon.ico 404**：新增国风favicon（红底金色"古"字），消除控制台404报错。
4. **版本号缓存**：所有资源版本号从v82统一升到v83，强制浏览器刷新缓存。

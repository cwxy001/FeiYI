# 非遗古镇 - 开发日志

## v80 (2026-07-03)

### 功能改动
1. **取消点击地块建造**：移除点击空地弹出建造面板的功能，只能通过建造按钮建造。新手引导步骤同步调整（步骤0改为引导点击建造按钮）。

### 美术优化
2. **地块统一古镇地砖**：所有地块从草地/泥土/青石板混合改为统一青石板纹理，营造古镇氛围。
3. **配色方案优化**：新增黛青色系（#2F4F4F），背景改为水墨渐变（米白→淡青），增加层次感。
4. **顶栏底栏木质纹理**：资源栏和底栏增加木纹纹理+鎏金边框，按钮改为圆角胶囊形。
5. **12生肖圆环关卡选择器**：分页卡片列表 → 圆环12扇形选择器。
   - 12个生肖扇形（鼠牛虎兔龙蛇马羊猴鸡狗猪）
   - 中心放无尽挑战和特殊挑战按钮
   - 底部预留"24节气·天干地支"扩展位
   - 未解锁生肖灰色锁定，当前生肖高亮
6. **弹窗水墨晕染动画**：所有面板/弹窗打开时有水墨扩散效果（scale+blur）。
7. **按钮hover涟漪**：底栏按钮hover时金色水墨涟漪扩散。
8. **关卡卡片悬浮**：关卡卡片hover上浮+金色阴影。
9. **资源数字弹跳**：经营模式资源数字变化时缩放高亮（仅经营模式生效）。

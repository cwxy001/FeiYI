## 一、项目概述

### 1.1 基本信息

- **游戏名称**：非遗古镇
- **游戏类型**：经营模拟 + 卡牌塔防 + 图鉴收集 + Roguelike
- **题材**：中国非物质文化遗产（剪纸、皮影戏、刺绣、陶瓷等12种非遗工坊）
- **目标平台**：H5网页（当前）→ 微信小游戏（后续迁移）
- **技术栈**：HTML + CSS + Vanilla JS + Canvas 2D + Web Audio API + Lottie
- **外部依赖**：Google Fonts（Noto Serif SC + Ma Shan Zheng）、Lottie-web CDN

### 1.2 核心玩法循环

```
经营界面（建造工坊/升级/产出资源/装饰）
       ↕ 双向联动
塔防闯关（卡牌构筑/放置防御塔/三合一升级/击败敌人）
       ↕ 数据互通
图鉴收集（解锁非遗/怪物/物质文化遗产）
       ↕ 持续成长
Roguelike遗物（局内增益三选一 + 永久遗物收集）
```

### 1.3 设计参考

| 参考游戏 | 借鉴要素 |
|---------|---------|
| 无悔华夏 | 国风题材、名臣/传承人系统、政策/科技树 |
| 部落冲突 | 经营↔塔防双向联动、阵型设计 |
| 遗弃之地 | 中式民俗美术风格、Roguelike局内Build |
| 保卫萝卜 | 休闲塔防、关卡渐进难度、海量关卡 |

### 阶段三：塔防战斗系统

**完成内容：**
- 关卡选择界面：12关卡十二时辰圆形排列
- 卡牌系统（CardSystem）：
  - 牌库构建：已建造工坊每种2张，Fisher-Yates洗牌
  - 初始抽5张手牌（不自动合并）
  - 手动抽卡：50人气/次，上限999
  - 自动合并：3张同名手牌→1张mergeLevel=2的卡（不递归）
- 防御塔系统（Tower类）：
  - 5级升级，每级伤害+20%、射程+10%
  - findTarget寻找最近敌人
  - 三合一升级：场上2个同级同名塔+手牌1张→消耗1个场上塔+手牌，升级另一个
  - 场上3个同级同名塔→自动三合一（不消耗手牌）
  - 塔防中所有塔初始1级，不受经营界面等级影响
- 敌人系统（Enemy类）：统一hp字段，沿路径移动，头顶血条
- 十二生肖BOSS：各有独特技能（召唤/冲撞/分裂/闪避/AOE/中毒/冲锋/治疗/分身/增益/三头/万象）
- 游戏控制：暂停/2倍速/退出，准备阶段10秒可跳过
- 胜负判定：主灯被摧毁=失败，清除全部波次=胜利
- 结算弹窗：胜利发奖励（DataIntegration.onLevelVictory）
- 难度系统：渐进式难度（前3关HP-30%~-10%，4-6关正常，7-12关HP+10%~+40%）

**关键文件：** `js/tower-defense.js`(3453行) `js/data-integration.js`(116行)
